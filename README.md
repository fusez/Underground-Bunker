# Underground Bunker Map
### This is a map made to showcase my [mapeditor](https://github.com/fusez/Map-Editor-V3).

## Features
* ~670 objects
* 15 actors.
* 2 vehicles.
* a fully functional elevator
* steel doors that open and close automatically
